## 📋 Vue d'ensemble

**US High LMS** est une plateforme éducative complète pour les élèves de 6ème à 12ème année avec:

✅ **Gestion pédagogique** - Cours, modules, examens  
✅ **Simulateur d'entreprise** - 20+ secteurs, gestion financière (XAF)  
✅ **IA intégrée** - Tuteur ChatGPT-4, correction automatique  
✅ **Multi-plateforme** - Web + Mobile Android  
✅ **Multilingue** - FR, EN, ES et plus  
✅ **Sécurité entreprise** - JWT, 2FA, chiffrement
